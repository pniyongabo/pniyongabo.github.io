Creating a Website

Creating a website using hugo is easy, start by following this quick start guide to setup and run the website in a local environment.

After running your website in local, you will have to decide which tool to use to host your static website. Github Pages and Netifly are free and recommended for a quick and east setup.

For more information and links about Hugo, check out the Hugo website or read the about hugo page.
