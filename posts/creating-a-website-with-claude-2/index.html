Adding AI Generation Features to AIRTIFACT: A Prompt-Driven Development Story

This post documents how we added AI image generation capabilities to AIRTIFACT.COM through structured prompting and iterative development with an AI assistant.

Initial Feature Discussion

Prompt 1: Feature Proposal

“lets go back to building. i wannt to dynamically generate images and poems using llm apis. can you help?”

Requirements Analysis

Prompt 2: API Selection

The AI assistant presented options for:

  1. Image Generation APIs
  2. Text Generation APIs
  3. Cost considerations

My Response

“let use free options for now. or cheap ones.”

Technical Implementation

Prompt 3: API Integration Choice

The AI suggested starting with:

  1. Stability AI for images (1000 free images/month)
  2. GPT-3.5-turbo for poems (cost-effective option)

My Response

“let s go with Stable Diffusion. do i need an account and an api key”

Implementation Process

Prompt 4: Initial Setup

“i have a key for stability.ai”

Prompt 5: Error Resolution

When encountering API key issues:

“getting this error: Error: Incorrect API key provided: ${proce*********************KEY}”

Prompt 6: Image Display Fix

“it is now working but the image is not getting displayed correctly. I think the issue is with the name”

Key Technical Solutions

  1. API Integration

    • Implemented Stability AI endpoint
    • Added proper error handling
    • Secured API key management

  2. File Management

    • Created safe filename generation
    • Implemented proper image storage
    • Added debug logging

  3. User Interface

    • Built admin generation interface
    • Added loading states
    • Implemented error displays

Effective Debugging Patterns

  1. Incremental Testing

    • Test API connection first
    • Verify file saving
    • Confirm image display

  2. Error Handling

    • Clear error messages
    • Detailed logging
    • User-friendly feedback

  3. Code Refinement

    • Simplified file naming
    • Improved error handling
    • Enhanced user feedback

// Key implementation features
function createSafeFilename(prompt) {
  return prompt
    .toLowerCase()
    .replace(/[^a-z0-9]/g, '-')
    .replace(/-+/g, '-')
    .replace(/^-|-\$/g, '')
    .slice(0, 50);
}
